# Projet LaTeX - Rapport de Science des Données

## Structure
- `main.tex` : fichier principal
- `rapportUDSDS.cls` : classe personnalisée du rapport
- `frontmatter/` : introduction et conclusion générale
- `chapitres/` : un fichier par chapitre (`chapitre_01.tex`, ..., `chapitre_05.tex`)
- `annexes/` : annexes éventuelles
- `logos/` : éléments graphiques de la couverture

## Compilation
Compilation recommandée avec **pdflatex** :

```bash
pdflatex main.tex
pdflatex main.tex
```

Si vous souhaitez un affichage complet des références croisées, lancez deux passes successives.

## Personnalisation rapide
Dans `main.tex`, modifier au besoin :
- le thème
- le nom du professeur
- l'année académique
- le niveau et la filière
- la table des membres du groupe
